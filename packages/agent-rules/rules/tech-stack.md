# Tech Stack Defaults

These are the default choices unless stated otherwise:

- **Runtime & package management:** Bun
- **Full-stack React apps:** Vite + TanStack Start
- **TanStack libraries by default:** Router, Query, Form, Table, Virtual, Pacer
- **Backend:** Convex
- **Linting:** oxlint with type-aware mode and `@zachsents/oxlint-config`; React projects also use `@zachsents/oxlint-config/react`
- **Formatting:** Prettier with `@zachsents/prettier-config`
- **Type checking:** tsgo, falling back to tsc
- **Styling:** TailwindCSS v4
- **Component primitives:** shadcn/ui, Base UI variant
- **Authentication:** Better-auth
- **Utilities:** Remeda, Zod v4, date-fns
